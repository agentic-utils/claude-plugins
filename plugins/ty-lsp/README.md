# ty-lsp

Python language server (ty) for Claude Code, providing static type checking and code intelligence.

## Supported Extensions
`.py`, `.pyi`

## Installation

Install ty globally:

```bash
uv tool install ty@latest
```

Or with pip:

```bash
pip install ty
```
